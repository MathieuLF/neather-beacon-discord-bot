## Politique v1

Le bot ne modifie pas automatiquement les icones, avatars ou bannieres.

Raisons:

- Discord limite certains visuels selon le niveau et les fonctionnalites du serveur.
- Les avatars des applications bots se gerent proprement depuis le portail Discord Developer.
- Les icones de roles peuvent dependre des options disponibles sur le serveur.

Ces fichiers servent donc de reference documentaire et de source locale stable pour les changements manuels.
